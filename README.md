# 学习笔记仓库

这是我的个人学习笔记仓库，里面记录了我从大学时期开始学习的所有知识，当然这个仓库会随着我的成长不断丰富，也希望这个仓库中的内容可以帮助到你。笔记使用 Markdown 格式编写，支持标准的 Markdown 查看器渲染 和 obsidian 查看。

## 内容概述

### ELF 文件格式
深入学习 ELF（Executable and Linking Format）可执行文件格式，涵盖：
- ELF 文件结构与类型
- Section 与 Segment 的区别
- PIE 与 ASLR 安全机制
- ELF 注入技术原理
- 动态链接与 PIC 原理
- System V AMD64 ABI 调用约定
- Binutils 工具链完全指南

### Bochs 模拟器
- Bochs 模拟器详细介绍

### QEMU
- QEMU User 模式深入研究
- 二进制翻译原理与架构解析

### 操作系统
系统学习操作系统核心概念：
- 进程管理与调度
- 内存管理（分页、分段、虚拟内存）
- 文件系统
- 设备管理与 I/O 系统
- 进程同步与通信

### 数据库原理
数据库系统理论与实践：
- 关系数据模型与 SQL 语言
- 关系规范化理论
- 数据库设计与 E-R 图
- 查询优化
- 事务与并发控制
- 数据库恢复技术

### 前端技术
CSS 相关知识笔记：
- CSS 属性计算过程
- 包含块与盒模型

### Git 版本控制
常用 Git 命令与工作流程参考。

### 微信小程序开发
微信小程序开发的学习路径与清单。

## 图片查看

本仓库中的图片通过相对路径引用，存放在各章节目录下的 `attachments/` 文件夹中。例如：

```
操作系统/
├── 1.引论.md
└── attachments/
    └── Pasted image 20260330131408.png
```

在支持相对路径的 Markdown 查看器中，图片可正常显示。
