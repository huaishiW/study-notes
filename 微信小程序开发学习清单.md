# 微信小程序开发学习清单

## 一、小程序基础认知

- [ ] 了解微信小程序是什么，与原生App和Web App的区别
- [ ] 了解小程序的发展历程和生态
- [ ] 了解小程序的适用场景（电商、工具、内容资讯、生活服务等）
- [ ] 阅读微信小程序官方文档概述部分

## 二、开发环境搭建

- [ ] 注册微信小程序账号（微信公众平台 https://mp.weixin.qq.com/）
- [ ] 了解小程序AppID的作用和获取方式
- [ ] 下载并安装微信开发者工具
- [ ] 熟悉开发者工具界面（模拟器、编辑器、调试器）
- [ ] 创建第一个小程序项目并运行
- [ ] 了解开发者工具的常用功能（真机调试、代码检查、构建npm）

## 三、小程序项目结构

- [ ] 理解小程序的项目目录结构
  ```
  ├── pages/           # 页面目录
  │   └── index/       # 页面文件夹
  │       ├── index.js
  │       ├── index.wxml
  │       ├── index.wxss
  │       └── index.json
  ├── utils/           # 工具函数
  ├── app.js           # 小程序入口文件
  ├── app.json         # 全局配置
  ├── app.wxss         # 全局样式
  └── project.config.json  # 项目配置
  ```
- [ ] 理解每个文件的作用
- [ ] 了解sitemap配置

## 四、WXML模板语法

- [ ] 掌握WXML与HTML的区别
- [ ] 掌握数据绑定（`{{}}`语法）
- [ ] 掌握列表渲染（`wx:for`、`wx:key`）

- [ ] 掌握模板引用（import/include）
- [ ] 掌握自定义组件的数据传递

## 七、常见组件

- [ ] 视图容器：view、scroll-view、swiper、movable-view
- [ ] 基础内容：text、icon、progress
- [ ] 表单组件：button、input、textarea、checkbox、radio、picker、slider、switch
- [ ] 导航：navigator
- [ ] 媒体组件：image、audio、video、camera
- [ ] 地图：map（需要申请权限）
- [ ] 开放组件：open-data、web-view

## 八、常用API

- [ ] 网络请求：wx.request()
- [ ] 本地存储：wx.getStorageSync/Async、wx.setStorageSync/Async
- [ ] 路由跳转：wx.navigateTo、wx.redirectTo、wx.switchTab、wx.reLaunch
- [ ] 交互反馈：wx.showToast、wx.showModal、wx.showActionSheet
- [ ] 页面滚动：wx.pageScrollTo
- [ ] 位置获取：wx.getLocation
- [ ] 扫码：wx.scanCode
- [ ] 分享：onShareAppMessage
